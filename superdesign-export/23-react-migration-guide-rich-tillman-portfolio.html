1. Environment Setup

Recommended: **Next.js 14+ (App Router)** for SEO and performance or **Vite** for SPA flexibility.

# Create Next.js App (Recommended)
npx create-next-app@latest rich-tillman-portfolio --typescript --tailwind --eslint

# Or Vite React
npm create vite@latest rich-tillman-portfolio -- --template react-ts

Dependencies

npm install lucide-react framer-motion clsx tailwind-merge iconify-icon three @react-three/fiber @react-three/drei
