— Overview

Bridging the gap between design intent and shipped code.

ForgeKit MCP is a Model Context Protocol server that integrates directly with AI coding assistants like Cursor, Windsurf, and Claude Desktop. It exposes Figma designs as structured context — allowing AI agents to generate pixel-accurate React components from design files without manual screenshot interpretation.

The server parses Figma's REST API, normalizes auto-layout constraints into flexbox equivalents, extracts design tokens (colors, typography, spacing), and feeds them to the AI as typed JSON schemas. The result: components that match designs on the first pass, not the tenth.

— Feature Highlights

Figma Node Parser

Deep-traverses Figma's node tree and normalizes auto-layout constraints into CSS flexbox equivalents — preserving responsive behavior.

React Codegen

Generates typed React + TypeScript components with Tailwind classes or CSS Modules. Outputs Storybook stories automatically.

Design Token Sync

Extracts colors, typography scales, spacing units, and shadows from Figma variables — outputs to style-dictionary format.

12 MCP Tools

Exposes tools like get_figma_file, extract_components, map_design_tokens, and generate_component — all typed with JSON schemas.

IDE Integrations

Native support for Cursor, Windsurf, Claude Desktop, and Continue.dev. Zero-config setup via npx forgekit-mcp.

Component Diffing

Compares generated components against existing design system entries — flags duplicates and suggests refactors before PR creation.
